3D Terrain Generation

My 3D Terrain Generator is an adaptation of Mewo2's fantastic map generation code (found here ) to be visualised in 3D. the project was designed to originally be used with a Arduino based controller but I've used datGUI to emulate the functionality. originally I was planning on writing the project in processing, but after having trouble porting Mewo2's javascript code I instead decided to stick with using javascript, p5.js, three.js, and a few other libraries to accomplish the same result
